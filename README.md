# ХолодСамара — FastAPI сайт услуг по обслуживанию кондиционеров

Простой учебный проект на FastAPI: лендинг с услугами, мастерами, отзывами, зоной работы вокруг Самары и формой заявки.

## Что внутри

- Главная страница `/`
- Статические картинки мастеров
- Блок услуг
- Блок отзывов
- Область работы вокруг Самары
- Контактный номер
- Простая форма заявки
- API:
  - `GET /api/services`
  - `GET /api/reviews`
  - `GET /api/coverage`
  - `POST /api/lead`

## Установка

```bash
python -m venv venv
```

Windows:

```bash
venv\Scripts\activate
```

Linux / macOS:

```bash
source venv/bin/activate
```

Установка зависимостей:

```bash
pip install -r requirements.txt
```

## Запуск

```bash
uvicorn app.main:app --reload
```

Открыть в браузере:

```text
http://127.0.0.1:8000
```

## Где поменять номер телефона

В файле:

```text
app/main.py
```

Найди строку:

```python
"phone": "+7 (999) 123-45-67",
```

и замени на нужный номер.

## Примечание

Форма заявки демонстрационная. Сейчас она возвращает ответ от backend, но не сохраняет данные в базу.
Позже можно добавить PostgreSQL, отправку заявок в Telegram или админ-панель.
